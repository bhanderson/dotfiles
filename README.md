##Installation:
To install, clone it somewhere. I use ~/code/dotfiles
	cd ~/code/dotfiles
	git submodule init
	git submodule update
	ln -s ~/code/dotfiles/vimrc ~/.vimrc
	ln -s ~/code/dotfiles/.vim ~/.vim

open up vim
press enter a few times then enter :BundleInstall
exit vim
